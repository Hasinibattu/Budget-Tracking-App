# Budget Tracking App

A desktop application for tracking budget and expenses, built with Electron, Node.js, and HTML/CSS/JS.

## Features
- Track incomes and expenses
- Manage user profiles
- Analyze budget flow

## Installation and Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Hasinibattu/Budget-Tracking-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Budget-Tracking-App
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Project Structure
- `login`, `main`, `welcome`: Application views and assets
- `emain.js` / `main.js`: Main process entry points for Electron
- `styles`: Application styles
- `users.json` / `backup.json`: App data storage
